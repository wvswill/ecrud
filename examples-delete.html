Delete

Example method Delete.

Documentation

This example uses Oracle HR schema.

Code

Method to delete based on a model.

//Method to delete based on a model.
using (var Ecrud = new ECRUD.OracleMapping())
{
    var Employee = Ecrud.Find("FIRST_NAME = 'WILLIAN' AND LAST_NAME = 'VALE'", DataBase.HR);

    var Deleted = Ecrud.Delete(Employee, DataBase.HR);
}

Code Async

Method to delete based on a model.

//Method to delete based on a model.
using (var ECRUD = new ECRUD.OracleMapping())
{
    var Employee = await ECRUD.FindAsync("FIRST_NAME = 'WILLIAN' AND LAST_NAME = 'VALE'", DataBase.HR);

    var Deleted = await ECRUD.DeleteAsync(Employee, DataBase.HR);
}
